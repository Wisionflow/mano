# Полный аудит — Mano

## Порядок запуска

```
1. audit-code     → безопасность данных и надёжность кода
2. audit-medical  → актуальность медицинского контента
```

Если на шаге 1 найден P0 (утечка данных) — остановиться.

---

## Исполнение

### 1. CODE AUDIT
Следуй инструкциям из `audit-code.md`:
- Безопасность данных пациента
- Telegram bot авторизация
- Claude API промпты
- OCR / Whisper надёжность
- ChromaDB целостность

### 2. MEDICAL CONTENT AUDIT
Следуй инструкциям из `audit-medical.md`:
- MEDICAL_SUMMARY.md актуальность
- /sos и /hospital карточки
- Лабораторный трекер
- Дневник здоровья
- Расхождения с ЕМИАС
- Промпты с медицинскими данными

## Правила

- Максимум 2 P0, 3 P1, 4 P2 — фокус на критическом
- Безопасность данных > актуальность контента > тех. долг
- Не дублируй детали из отдельных аудитов
- Один вопрос в конце — самый неудобный
